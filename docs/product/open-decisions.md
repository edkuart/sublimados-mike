# Decisiones pendientes antes de programar

## Negocio

- Pais y moneda principal.
- Nombre comercial exacto y datos legales.
- Numero oficial de WhatsApp.
- Horarios de atencion.
- Metodos de entrega: envio, recogida, ambos.
- Zonas iniciales de cobertura.
- Politica de anticipo si no habra pago online.

## Catalogo inicial

- Categorias de lanzamiento.
- Productos exactos de primera version.
- Precio base por producto.
- Variantes por categoria.
- Campos de personalizacion por producto.
- Tiempos de produccion por producto.

## Personalizacion

- Archivos permitidos: JPG, PNG, PDF, SVG, AI, PSD.
- Tamano maximo de archivo.
- Cantidad maxima de archivos por cotizacion.
- Si se aceptan fotos de baja calidad bajo responsabilidad del cliente.
- Si siempre hay aprobacion de diseno antes de producir.

## Autenticacion

- Permitir cotizar como invitado.
- Requerir cuenta para historial.
- Activar Facebook Login en Fase 3 o dejarlo para despues.

## Arquitectura

- Monolito Next.js inicial o frontend/backend separado desde el inicio.
- Supabase Storage, Cloudinary o combinacion.
- Vercel + Supabase como infraestructura inicial.

## Politicas

- Cambios y devoluciones para productos personalizados.
- Cancelaciones antes y despues de aprobacion de diseno.
- Uso de imagenes del cliente.
- Retencion y borrado de archivos.
- Privacidad y consentimiento.

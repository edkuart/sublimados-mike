# Catalogo inicial y taxonomia

## Criterio de organizacion

El catalogo debe organizarse por familias comprensibles para el cliente y por atributos
utiles para produccion. La regla base es:

- Categoria: como navega el cliente.
- Producto: articulo vendible.
- Variante: diferencia de precio, inventario o produccion.
- Personalizacion: informacion unica del pedido del cliente.

## Categorias iniciales recomendadas

Productos posibles:

- Taza blanca 11 oz.
- Taza magica.
- Taza con interior/color.

- Taza con caja regalo.

- Capacidad.
- Color interior.
- Tipo de taza.
- Acabado.

Personalizacion sugerida:

## Campos de personalizacion reutilizables

- Nombre.
- Texto principal.
- Frase secundaria.
- Fecha.
- Foto.
- Logo.
- Archivo de referencia.
- Fuente/tipografia.
- Color de texto.
- Posicion deseada.
- Observaciones.

## Filtros publicos iniciales

- Categoria.
- Rango de precio.
- Producto para: regalo, empresa, evento, pareja, cumpleanos.
- Color base.
- Material.
- Tiempo estimado de produccion.
- Personalizable con foto.
- Personalizable con logo.

## Recomendacion operativa

Lanzar con pocas categorias bien cuidadas:

1. Tazas.
2. Playeras.
3. Termos.
4. Llaveros.
5. Cojines.
6. Regalos personalizados.

Agregar gorras, decoracion y productos especiales despues de validar carga operativa y
calidad de fotografias.
